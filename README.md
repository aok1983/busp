# busp

Структура проекта

lib        - каталог для вспомогательных библиотек

libnemsim  - корневой каталог библиотеки модельного программного обеспечения
	include - каталог заголовочных файлов модельной библиотеки (описание интерфейса библиотеки)
	sources - корневой каталог файлов с исходными текстами
		disp    - диспетчер модели 
		include - заголовочные файлы для внутреннего использования в модели (внутренные интерфейсы)
		sims    - каталоги моделей
			busp - исходные тексты модели БУСП
			suba - исходные текты модели СУБА
		support - вспомогательное ПО

msvc90     - файлы проекта в Microsoft visual studio 2008 (9.0)

simcore    - ядро модели для обеспечения тестирования
	include - каталог заголовочных файлов ядра модели. Интерфейсы предоставляемые для модели со стороны платформы.
	sources - каталог исходных текстов ядра

test       - тесты cmocka для проверки библиотеки модельного ПО
	include - каталог загловочных файлов для тестов (Интерфейсы для использования в тестах)
	sources - блочные тесты библиотеки модельного ПО

.gitignore - служебный файл Git для настроек игнорирования
